# 🧩 Crossword Puzzle Game (C++)

## 📌 Overview
This project is a console-based **Crossword Puzzle Game** developed in C++ as part of Data Structures and Algorithms (DSA). It uses structured data and algorithmic logic to generate and manage crossword puzzles.

The game utilizes a dataset of words and clues to create puzzles dynamically and allows users to interact with the grid through a simple console interface.

---

## 🚀 Features
- 🧩 Generate crossword puzzle from dataset  
- 📖 Display crossword grid  
- ✍️ Input answers for clues  
- ✅ Validate user answers  
- 🔍 Search words within dataset  
- 📂 Dataset integration (words & clues)  
- 🖥️ Console-based gameplay  

---

## 🛠️ Technologies Used
- **C++**
- **Data Structures & Algorithms (DSA)**
- Arrays / Strings / Grid-based logic
- File Handling (for dataset)
